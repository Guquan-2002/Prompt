# Prompt
```
这是对一个病例的分析步骤：
1. 搜集临床资料：逐字逐句提取病例中的病史要点和临床表现，包括现病史、既往史、症状体征、体格检查、辅助检查、个人情况、婚姻情况、家族情况等内容，不论是阳性的或是阴性，不论是重要信息或是无关信息，都需要逐个提取。
2. 综合分析评价：分析解释所有的病史要点和临床表现背后可能提示的信息，结合多个学科的知识（包括但不限于解剖学、组织学、胚胎学、生理学、生物化学、分子生物学、病理生理学、免疫学、诊断学、检验学、影像学、病理学、麻醉学、预防医学、流行病学、内科学、外科学、药理学）全面分析其背后的每一种可能性，阐述其对诊断、鉴别、分型、治疗的提示作用、指导作用、限制作用、排除作用。
3. 指导临床诊治：根据病史要点、临床表现、综合分析，针对此病例给出你的诊断（病例若存在多种疾病，则应给出多诊断，诊断需参考"ICD-10"）、拟诊讨论（包括诊断依据、鉴别诊断，诊断依据至少3点，鉴别诊断至少3条）、诊疗计划（包括后续的检查、后续的治疗、护理要求、注意事项、不应采取的行为、需提示的风险）。
---
请按照这个步骤，对接下来的病例进行分析。
```
